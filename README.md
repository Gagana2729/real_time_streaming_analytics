# real_time_streaming_analytics# Real-Time Streaming Analytics Pipeline

## Overview

This project processes high-volume event streams in real time using:

- Apache Kafka
- PySpark Structured Streaming
- Delta Lake
- Elasticsearch
- Streamlit Dashboard

It computes live KPIs such as:

- Events per second
- Revenue per minute
- Active users
- Top pages
- Error rate
- Anomaly detection alerts

End-to-end latency: < 5 seconds
